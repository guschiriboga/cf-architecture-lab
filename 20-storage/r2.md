# R2 (Object Storage)

**Free plan headline (monthly):** 10 GB-month storage, 1M Class A ops, 10M Class B ops, **egress to Internet free** (Standard tier).  
Sources:
- Pricing: https://developers.cloudflare.com/r2/pricing/
- Limits: https://developers.cloudflare.com/r2/platform/limits/

## Summary
S3-compatible object storage with egress to the public Internet priced at $0 (per docs). Good for assets, backups, and large objects.

## Free Tier
- **Storage:** 10 GB-month
- **Class A ops:** 1,000,000 / month
- **Class B ops:** 10,000,000 / month
- **Egress:** Free to Internet (Standard only)

## Notes / Gotchas
- Free tier applies to **Standard** storage (not Infrequent Access).
- Object size limit ~4.995 TiB; single-part upload ~4.995 GiB.
